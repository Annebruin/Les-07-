Wat is mijn ervaring met HTML?

Wat Heb Ik Geleerd?

Bij HTML heb ik vooral veel geleerd
over het maken en ontwerpen van websites
in CSS en HTML!

Ik heb vooral geleerd hoe ik een HTML
bestand moet opzetten en hoe ik dit bestand
zo goed mogelijk kan organiseren en onderhouden.

Mijn Actiepunten

Mijn actiepunten liggen vooral bij mijn zwakke
punten als ontwerper. Ik vind het belangrijk om
voor mezelf meer te gaan letten op het leren van
nieuwe technieken, het opzoeken van dingen die
ik nog niet weet, het ordenen van mijn HTML bestanden,
het controleren van mijn HTML zodat alles goed staat,
en de trail en error die daarbij komt kijken.

Mijn Ontwerp

Ik vind HTML persoonlijk erg moeilijk, dus als ik
dit ontwerp kan uitwerken in HTML, ben ik al
supertrots op mezelf!
